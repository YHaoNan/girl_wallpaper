# 关于仓库
就是一个老色批用来收集好看的美女壁纸的仓库，我找到就上传收藏起来。

# 格式
`horizontal`文件夹中存储的是横向的图片，适用于电脑、平板电脑等设备。

`vertical`文件夹中存储的是纵向的图片，适用于手机、纵向平板电脑等设备。

目前文件夹中没有子文件夹，日后不知道。
